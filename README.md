# autorap

Full Automated Rap Singer

## PREAPRE

- Download keywordlist_furigana.csv
  - http://developer.hatena.ne.jp/ja/documents/keyword/misc/catalog
- Prepare Yahoo's Application ID
  - http://developer.yahoo.co.jp/webapi/jlp/

# RUN

% bundle install
